# SeekDesk Daily AI Ecosystem Direction

## Product Position

SeekDesk is a practical AI workspace for everyday work. It should help users complete recurring knowledge-work tasks through chat, templates, connected context, and safe workflow automation.

SeekDesk should remain compatible with two app modes:

- `daily_work`: the active product mode for the current implementation.
- `coding_agent`: a reserved compatibility mode for future development workflows.

The current milestone only develops `daily_work`. Coding-agent UI and tool execution should not be exposed as a primary surface yet.

## Primary Jobs

- Draft and polish emails, reports, proposals, and posts
- Research topics and compare options with traceable notes
- Summarize meetings and turn notes into decisions and tasks
- Organize personal or team knowledge into reusable context
- Plan work across calendar, documents, and communication tools

## MVP Priorities

1. Keep the streaming AI chat stable and approachable.
2. Add task templates for common daily workflows.
3. Add a local/session knowledge area before broad external connectors.
4. Add explicit user approval before reading private documents or calling external services.
5. Track workflow outputs as reusable artifacts.

## Out Of Scope For The Current Daily-Work Milestone

- Full IDE replacement
- Shell command execution as a primary feature
- Git/diff workflows as top-level product surfaces
- Coding-agent autonomy beyond reserved contracts and compatibility checks
